class moldesign.units.quantity.MdtQuantity[source]¶

Bases: pint.unit.Quantity

This is a ‘patched’ version of pint’s quantities that can be pickled (slightly hacky) and supports more numpy operations. Users should never need to instantiate this directly - instead, construct BuckyBall quantities by multiplying numbers/arrays with the pre-defined units

Examples

>>> 5.0 * units.femtoseconds
>>> [1.0,2.0,3.0] * units.eV
